# Deloitte Data Analytics Job Simulation

This project is based on a virtual job simulation by Deloitte (Forage), where real-world business problems were solved using data analysis and visualization.

---

## 📊 Task 1: Data Analysis (Tableau)
- Built an interactive dashboard to analyze machine downtime
- Identified factory with highest downtime
- Analyzed device types contributing to failures
- Used filtering to explore factory-level insights

---

## 📈 Task 2: Forensic Technology (Excel)
- Analyzed employee compensation data
- Classified equality scores into:
  - Fair
  - Unfair
  - Highly Discriminative
- Used Excel formulas for automation

---

## 🛠 Tools Used
- Tableau Public
- Microsoft Excel

---

## 📁 Files Included
- Dashboard Screenshot (Task 1)
- Equality Table Excel File (Task 2)

---

## 📌 Key Insights
- Identified factory with maximum machine downtime
- Highlighted device types with frequent failures
- Detected inequality patterns in compensation data
